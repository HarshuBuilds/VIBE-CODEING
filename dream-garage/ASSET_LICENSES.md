## Important Notes

### License Compliance

1. **Always check the license** of each asset before using it in your project.
2. **Keep attribution** where required by the license.
3. **Do not use copyrighted assets** without proper permission.
4. **Respect the terms** of each license.

### Asset Optimization

All assets used in this project should be optimized for web performance:

- **3D Models**: Use Draco compression, reduce polygon count, optimize geometry
- **Textures**: Compress to WebP or AVIF, resize to appropriate dimensions
- **Sounds**: Convert to MP3 or OGG, optimize bitrate
- **Images**: Use Next.js Image component with optimization

When searching for assets:
